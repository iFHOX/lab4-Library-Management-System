# lab4-Library-Management-System
Lab4 for COSC236 - Members: Daniel Cuevas, Shondel Hayles, Simon Velez

This project demonstrates the use of GRASP principles in a simple Library Management System

## Principles Applied
- Information Expert
- Creator
- Controller
- Low Coupling
- High Cohesion

## How to Run
Compile and run `LibraryApp.java`.
